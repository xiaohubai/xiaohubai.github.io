面试-矩阵连乘

2017-03-28

动态规划分析：

  • 什么问题可以用动态规划?

（1）最优子结构
（2）子问题重

  • 分治法与动态规划的主要区别？

（1）分治法需要运行大量的重叠子问题。
（2）分治法采用自顶向下递归遍历。
（3）动态规划采用自底向上的方法，每次选择最优子结构，可以有效避免子问题重叠问题。

  • 动态规划解题过程？

（1）分析最优解的结构特征。
（2）建立最优解的递归式。
（3）自底向上。
（4）构造最优解。

问题描述：

给定n个矩阵的链<A1,A2,…..An>,Ai的规模为p[i]-1Xpi,求完全括号化方案，使得乘积A[1]A[2]….A[n]所需的标量乘法次数最少。

算法分析：

  • 找最优子结构特征

假设A[1]….A[j]的最优括号方案在A[k]和A[k+1]之间（i<=k<j） (A[i]….A[k])
(A[k+1]….A[j]) (k为1，2….j-1中的任意一值)。

  • 建立最优解的递归式

设m[i,j]为A[1]….A[j]的最小乘次。则m[i,j]=m[i,k]+m[k+1,j]+p[i-1]p[k]p[j]

  • 自底向上分析

对于子链A[i…k]存在k-i+1小于j-i+1个矩阵的积，子链A[K+1…j-1]也存在j-k小于j-i+1个矩阵的积。因此算法应该用长度递增的顺序求解问题。

  • 构造最优解

记录每个递增顺序最优括号方案的位置。

程序源码：

#include<cstdio>
#include<cstring>
#include<iostream>
using namespace std;
const int msize = 100;
int p[msize];
int m[msize][msize];
int s[msize][msize];
int n;

void matrixchain()
{
    int i,j,r,k;
    memset(m,0,sizeof(m));
    memset(s,0,sizeof(s));
    for(r = 2; r <= n; r++)        
    {
        for(i = 1; i <= n-r+1; i++)
        {
           j = i + r - 1;
           m[i][j] = m[i+1][j] + p[i-1] * p[i] * p[j]; 
           s[i][j] = i;                    
           for(k = i+1; k < j; k++) 
            {
                int t = m[i][k] + m[k+1][j] + p[i-1] * p[k] * p[j];
                if(t < m[i][j])
                {
                    m[i][j] = t;
                    s[i][j] = k;
                }
            }
        }
    }
}
void print(int i,int j)
{
    if( i == j )
    {
       cout <<"A[" << i << "]";
       return ;
    }
    cout << "(";
    print(i,s[i][j]);
    print(s[i][j]+1,j);
    cout << ")";
}
int main()
{
    cout << "请输入矩阵的个数 n:";
    cin >> n;
    int i ,j;
    cout << "请依次输入每个矩阵的行数和最后一个矩阵的列数:";
    for (i = 0; i <= n; i++ )
        cin >> p[i];
    matrixchain();
    print(1,n);
    cout << endl;

    cout << "最少乘法次数： " << m[1][n] << endl;
}
