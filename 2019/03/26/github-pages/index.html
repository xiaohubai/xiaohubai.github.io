color

文章摘要

mathjax

请按需开启，因为此项需要加载额外的 js 文件。

配置Github

  • 配置默认用户信息
  • 在github官网，”new repository”,名称”youname.github.io”

配置SSH KEY

1.查看是否已有SSH KEY

2.创建SSH KEY

3.记事本将C盘用户名下的隐藏的id_rsa.pub 复制到远端GitHub，用户名下的setting->SSH and GPG keys 下
4.确认是否成功

$ssh -T git@github.com

上传博客

1.打开 hexo/_config.yml，添加：

deploy:
    type: git 
    repo: https://版本库.git
    branch: master

2.安装Hexo git插件

$npm install hexo-deployer-git --save

3.发布到远端

$hexo d
