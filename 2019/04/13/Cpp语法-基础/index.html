2019-04-13

内存模型

  1. 32位的操作系统，内存空间中每一个字节的编号是一个32位二进制数。
  2. 内存编号从0000 0000 0000 0000 0000 0000 0000 0000至1111 1111 1111 1111 1111 1111 1111 1111。
  3. 16进制也就是0x00000000至0xFFFFFFFF。
  4. 10机制就是从0~2^32-1。
  5. 64位操作系统，内存编号也就是从64个0至64个1。

指针与指针变量

  1. 指针就是内存地址、内存编号。
  2. 指针变量：存储指针的变量、存储内存地址（内存编号）的变量。
  3. 指针变量和整型变量、字符型变量以及其他数据类型的变量一样都是变量类型，但不管什么类型的指针都是一样的，所占的内存大小都是四个字节（32位机器下），存储的是32位的内存地址。
  4. 对于int型指针变量，我们需要从该指针变量存储的（首）地址开始向后一直搜索4个字节的内存空间。
  5. 当我们使用*pa，必须先知道pa是一个整型的指针。
  6. 对指针变量进行算术运算，比如pa + 2,pb + +之类的操作，是以数据类型大小为单位的。
  7. pa + 2,相当于0x12ff60 + sizeof(int) * 2 = 0x12ff60 + 4 * 2 = 0x12ff68,不是0x12ff60 + 2。

指针变量/指针常量

  • 指针变量
  1. 指针变量首先是一个变量，指针变量存储了某个变量的内存首地址，我们通常认为”指针变量指向了该变量”。
  2. 某一时刻指针变量pa指向变量a，下个时候可能不存储变量a的首地址，而是存储变量c的首地址。也就是说指针变量是一个变量，它的值可以变动的。
  • 指针常量
  1. 存储固定的内存单元地址编号的”量“，它一旦存储了某个内存地址以后，不可再改存储其他的内存地址了。
  2. 指针常量对应的是const关键字定义的量，而不是指针字面量。
  3. &a,&b,&a + 2等是指针字面量。const int *p = &a;中的p才算是真正的指针常量
