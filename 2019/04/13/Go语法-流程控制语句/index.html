Go语法-流程控制语句

2019-04-13

for

  • 如同C的for循环
  • 省略若干条件

for…range 遍历

while

没有初始值、步长，只有条件。

没有 do-while 形式。

  • 如同C的while循环

死循环

  • 如同C的for(;;)循环

if

无需小括号 ( ) ，而大括号 { } 则是必须的

if else

switch

Go 自动提供了在这些语言中每个 case 后面所需的 break 语句。 除非以 fallthrough 语句结束，否则分支会自动终止.

case 不必为常量

  • switch 求值顺序

从上到下顺次执行，直到匹配成功时停止。

  • 没有条件的switch

没有条件的switch 和if-else-if-else 效果一样。更简洁

defer

defer 语句会将函数推迟到外层函数返回之后执行。

推迟调用的函数其参数会立即求值，但直到外层函数返回前该函数都不会被调用。

  • defer 栈

推迟的函数调用会被压入一个栈中。当外层函数返回时，被推迟的函数会按照后进先出的顺序调用。
