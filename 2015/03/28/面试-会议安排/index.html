面试-会议安排

2015-03-28

问题描述：

在有限的时间内召开更多的会议（任何两个会议不能同时进行）

算法设计：

(1) 初始化：将n个会议的开始时间，结束时间存放在结构数组中，按结束时间从小到大排序，结束时间相等时，按开始时间的从大到小排序。
(2)根据贪心策略选择第一个具有最早结束时间的会议，用last记录刚选中会议的结束时间。
(3)选择第一个会议后，依次从剩下未安排的会议中选择，如果会议i开始时间>=最后一个选中的会议的结束时间last,那么会议i与上一个已安排的会议相容，可以安排会议，否则继续遍历。

代码实现：

#include<iostream>
#include<algorithm>
using namespace std;
struct Meet {
    int u;
    int v;
    int num;
}meet[100];
bool cmp(Meet x, Meet y)
{
    if (x.v == y.v)
        return x.u > y.u;
    else
        return x.v < y.v;
}
int main()
{
    int n;
    cin >> n;
    for (int i = 0; i < n; i++)
    {
        cin >> meet[i].u >> meet[i].v;
        meet[i].num = i + 1;
    }
    sort(meet, meet + n, cmp);
    for (int i = 0; i < n; i++)
    {
        cout << meet[i].num  <<"    "<<  meet[i].u <<"    "<<  meet[i].v<<endl;
    }
    cout << meet[0].num << endl;
    int last = meet[0].v;

    for (int i = 1; i < n; i++)
    {
        if (meet[i].u > last)
            cout << meet[i].num << endl;
        last = meet[i].v;
    }
    int s;
      cin >> s;
}
